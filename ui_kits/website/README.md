# VP Buys Homes — Marketing Site UI Kit

Hi-fi recreation of the VPBuysHomes.com marketing site using the real brand guidelines.

## Components

- `Logo.jsx` — canonical VP + house + wordmark lockup. Supports `reversed` + `showWordmark`.
- `Header.jsx` — sticky nav, phone, amber CTA, scroll-aware border
- `Hero.jsx` — navy full-bleed, headline with amber accent word, lead-form card with submit state
- `TrustBar.jsx` — 4-stat row with amber accent bars
- `HowItWorks.jsx` — 3-step process, oversized step numerals
- `Benefits.jsx` — why-us list with navy photo placeholder
- `Situations.jsx` — 6 common seller scenarios
- `Areas.jsx` — service area list + open-territory card
- `Testimonials.jsx` — navy surface, 3 quotes with amber top accent
- `FAQ.jsx` — accordion with animated + → × icon
- `FinalCTA.jsx` — closing navy section
- `Footer.jsx` — dark, 4-col, with amber phone chip

## Open in `index.html`

The components are stitched together as a full click-thru marketing site demo. The hero offer form has a working submit → success state.
